# Tec-NOtes

This repository is a personal knowledge base for learning notes.

## Structure

- `notes/README.md` — notes index
- `notes/_template.md` — template for adding new notes

## How to use

1. Create a new file in `notes/` (for example, `notes/git-basics.md`).
2. Copy the contents of `notes/_template.md`.
3. Fill in the topic, summary, and key points.
4. Add the new file to `notes/README.md`.
